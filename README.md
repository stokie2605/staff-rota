# Staff Rota Management System
> **The 1-Line Mission:** Full-stack scheduling engine that automates shift assignment and prevents scheduling conflicts for medical, logistical, and call centre teams.

### ⚡ Quick Signal
* **The Problem:** Manual schedule allocation is highly prone to overlapping double-bookings, compliance failures (rest hours), and communication delays with employees.
* **The Solution:** A full-stack application featuring a FastAPI backend with SQLite double-booking constraint logic, a dockerized distribution structure, and a highly interactive React calendar dashboard.
* **The Tech:** `FastAPI` `React` `SQLite` `Docker`
